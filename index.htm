Alpine Linux Setup

Una guida essenziale per formattare, installare e configurare il tuo ambiente di sviluppo da zero.

1. Preparazione ISO

  • Download: Scarica la ISO di Alpine Linux (versione Standard o Extended) dal sito ufficiale.
  • Media: Flash della ISO su chiavetta USB usando Rufus o BalenaEtcher.

2. Accesso BIOS/UEFI

  • Riavvia il computer e premi ripetutamente F2, F12 o Canc.
  • Disabilita il Secure Boot nelle impostazioni di sicurezza.
  • Sposta la chiavetta USB in cima alla Boot Priority.
  • Salva con F10 e riavvia.

3. Installazione OS

Esegui il login come root (nessuna password) e digita:

setup-alpine
  • Seleziona il layout tastiera (es. it).
  • Configura la rete via DHCP.
  • Scegli il disco (es. sda) e scrivi sys per l'installazione su disco.

4. Ambiente Sviluppo

Docker (Consigliato)

apk update
apk add docker
rc-update add docker default
service docker start

Anaconda (Miniconda)

apk add wget bash
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
bash Miniconda3-latest-Linux-x86_64.sh
